# Home Network Simulation
This project simulates a basic home network using Cisco Packet Tracer. 
It demonstrates wired and wireless connections, DHCP configuration, and device communication.
## Features
- Wired connection between PC and Router
- Wireless connection for Laptop and Smartphone
- DHCP configuration for automatic IP assignment
- Ping tests to verify connectivity
## How to Test
1. Open the .pkt file in Cisco Packet Tracer
2. Check IP configurations for each device
3. Use Command Prompt to ping devices
## What I Learned
- Basic network setup using Cisco Packet Tracer
- Understanding of wired and wireless connections
- DHCP setup and IP assignment
- Verifying connectivity with ping command
